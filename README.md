# COMBINE

AI character-creation course (AI Education, Visuals, Growth) and its marketing site.

The course teaches one thing: **character consistency**, getting the same character from image to image and later frame to frame, taught across increasingly powerful roads.

## Layout

- `_planning/` the master course plan (single source of truth for structure, tiers, and build status)
- `00-foundations/` Level 0, Foundations (free): why consistency is hard (concept only)
- `10-cloud-path/` Level 1, Cloud Path (first paid): online tools, prompting, image-to-image
- `20-local-path/` Level 2, Local Path (Pro): local models, datasets, LoRA training
- `30-video/` Level 3, Video (top tier): consistency in motion, plus the cinema scene-craft layer
- `design-system/` brand design system (COMBINE)
- `assets/` logos and design assets
- `web/` static landing page (fallback)
- `web-react/` React + Vite landing page (current)

## Status

Levels 0 to 3 are drafted. `[FILL LATER]` markers across the lessons mark where deeper detail, examples, tool rosters, and video go during production. See `_planning/course-master-plan.md` for the live build status.
