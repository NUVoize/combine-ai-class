# Lesson 1.2: A Tour of the Cloud Tools

Lesson 1.1 gave you the three buckets: generation, edit, and hybrid. This lesson puts real tools in your hands and sorts each one into its bucket, so you stop guessing which app to open for a given job.

One warning before we start, and it matters: **this space moves fast.** New tools launch every month, old ones add features, and a generation-only app can wake up one morning with an edit mode bolted on. If you memorize a fixed list, it goes stale. So we are going to do two things at once. You will get the current roster, the tools worth your time right now, and you will get a way to *read* any tool you meet later so you can sort it yourself.

## How to read any tool in thirty seconds

Before the names, here is the lens. When you land on a new image tool, ask four questions in order:

1. **Can I hand it an image, or only words?** If there is nowhere to upload a picture, it is a generation model, full stop. That is the single most important question, because it decides whether the tool can hold a character at all (Lesson 1.1).
2. **If it takes an image, what does it do with it?** Does it edit that exact image, keeping the identity while changing the surroundings, or does it just take loose "inspiration" and invent something vaguely similar? Those are very different, and the second one drifts.
3. **What is it actually good at?** Speed, photorealism, stylization, text rendering, faces. Every tool has a personality, and fighting its personality is wasted effort.
4. **What does it cost me, in money and in friction?** Free with a watermark, credits, a subscription, a queue, a login. Friction decides what you will actually reach for day to day, not what looks best in a demo.

Run those four on anything and you will know where it sits and whether it earns a place in your workflow.

## Why we sort tools instead of ranking them

Notice we are not crowning a single "best" tool. There is no best, there is only *best for this job.* A generation model that invents gorgeous concept art is the wrong tool the moment you need that same character again, and the friendliest hybrid is overkill when you just want a fresh empty background. Sorting beats ranking, because the job changes from shot to shot.

That idea, matching the tool to the job, is exactly what the next lesson is about.
