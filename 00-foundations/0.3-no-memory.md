# Lesson 0.3: Every Image Is an Island

This is the shortest lesson in the module, and the most important one to never forget. We have circled it from a couple of angles already. Now let's state it flat out and let it stand on its own.

**The model has no memory. Every image it makes is an island.**

The picture it generated one second ago does not exist as far as the next generation is concerned. It cannot see it. It cannot refer back to it. It has no sense that it ever made it. There is no "the last one," no "again," no "like before," not because the model is ignoring those words, but because there is genuinely nothing on the other side of them. Each press of *generate* is the first and only image that instance of the model has ever been asked to make.

This is why character consistency is a *skill* and not a *setting*.

If the model remembered, this whole course would be one sentence: "make the same person again." There would be nothing to teach. The entire reason there is a craft here, training datasets, edit models, LoRAs, all of it, is that you are constantly working *against* a machine that forgets everything the instant it finishes.

So whenever something goes wrong later, the face drifts, the outfit changes, the character will not hold, your first instinct should snap back to this lesson. The model did not *lose* your character. It never had one to lose. It only ever knows what you put in front of it *this* time.

Tape that to the wall. Everything else is just methods for putting the right thing in front of it.
